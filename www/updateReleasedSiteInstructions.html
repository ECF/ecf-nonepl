Install via Update Site -- for use with Eclipse 3.5

  • In Eclipse 3.5/Galileo, choose Help -> Install New Software...
  • Click Add
  • Type a Name of yur choosing
  • For the URL, put in http://ecf1.osuosl.org/OSUrelease-repo/repo
  • Click OK.
  • Select the Extras you want. Do not select both the Win32 and the Linux versions of Skype. If you are running Linux, never select the Win32 version of Skype.
  • Follow through the remaining dialog to complete the installation.

Using the Skype Plugin on Windows XP

Using the Skype plugin is straightforward, but setting it up is not intuitive. This is a one-time setup.

To use the Skype plugin with ECF, you must have Skype installed on your computer. This installation is distinct from any Eclipse or Eclipse plugin installation. To install Skype, go to http://www.skype.com Download and install Skype. Make yourself an account.

  • Install ECF. The directions on http://www.eclipse.org/ecf/downloads.php
  • Then, install the Skype plugin. Choose the ECF Skype Provider (win32) from http://ecf1.osuosl.org/OSUrelease-repo/repo
  • Start up Skype and sign in. This invocation is distinct from invoking Eclipse.
  • Start up Eclipse. Dropdown the ECF menu and choose Skype. On Win XP you'll see a small popup window appear in the lower right. Click on the Click on this to accept/deny.
  • Now from within the Skype window, click on the Tools tab and choose Options.
  • Click on Advanced.
  • Then, at the bottom of the screen that appears, click on Manage other programs' access to Skype.
  • In the window that appears, click on Change.
  • In the window that appears, click on Allow this program to use Skype. Click on OK.
  • Another window appears. Its title is New Skype COnnection. This is the window that will always appear when you choose Skype from the ECF dopdown menu. Up until now the Eclipse window itself has been waiting for Skype to be configured.
  • Click on Finish. In Eclipse, in the Contacts window, click on the + sign beside your Skype account to see your contacts.

The file UsingTheSkypeECFpluginOnWinXP.pdf contains the same instructions as above, but with screenshots.

Using the Skype Plugin on 32-bit Linux

To use the Skype plugin with ECF, you must have Skype installed on your computer. This installation is distinct from any Eclipse or Eclipse plugin installation. To install Skype, go to http://www.skype.com Download and install Skype. Make yourself an account.

  • Install ECF. The directions on http://www.eclipse.org/ecf/downloads.php
  • Then, install the Skype plugin. Choose the ECF Skype Provider (linux) from http://ecf1.osuosl.org/OSUrelease-repo/repo
  • Start up Skype and sign in. This invocation is distinct from invoking Eclipse.
  • Start up Eclipse. Dropdown the ECF menu and choose Skype.
  • The "Skype Connection Wizard" window appears. This window requests that you confirm your connection to Skype. Click OK.
